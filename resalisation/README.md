# Blog Database Realization

This project contains the SQL schema for a blog database system.

## Description

The database includes tables for:
- Users (user information)
- Categories (post categories)
- Posts (blog articles)
- Comments (post comments)

## Prerequisites

- MySQL or compatible database system

## Setup

1. Connect to your MySQL server
2. Run the `Realisation_Blog_DB.sql` file

The script creates the database, tables, and inserts sample data.
